- **Tipografía premium**: Bebas Neue, Poppins, Montserrat

- **Animaciones suaves** con fade-in, scale, y slide effects
- **Efectos glassmorphism** en cards y overlays
- **Gradientes dinámicos** con colores técnicos
- **Hero full-screen** con imagen de fondo y overlays
